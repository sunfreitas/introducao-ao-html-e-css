Introdução

Abra o arquivo desta página em um editor de códigos, como o Sublime Text 3, ou aperte as teclas CTRL+U para visualizar o código fonte.

Para visualizar o resultado desta página, abra seu navegador preferido. Aperte a combinação de teclas CTRL+O para buscar pelo arquivo no seu sistema operacional.

Entendendo o HTML

!Doctype

Note que no início do documento você encontra a seguinte linha:

<!DOCTYPE html>

A declaração !Doctype informa ao navegador qual a versão do HTML você utilizou para criar o documento. Caso queira saber mais sobre ela visite sua página na Wikipédia.

A tag <html>

Logo abaixo do doctype abrimos a tag html. Ela representa a raiz do documento, onde todas as demais tags serão colocadas dentro.

A tag html deve ser aberta com <html> e fechada com </html>. Basta olhar para o código fonte desta página para ver como as demais tags estão 'aninhadas' dentro da html.

A tag <body>

Dentro da tag html, definimos a tag body. Ela representa todo o conteúdo do documento HTML. Assim como a tag html, só pode haver uma tag body dentro de um documento HTML.

Esta tag também deve ser aberta e fechada de forma correta utilizando <body> e </body>. Mais sobre a tag body no MDN.

Cabeçalhos

O título deste tópico foi colocado entre as tags <h3> e </h3>

Esta tag representa o 3º de 6 níveis das tagas h1-h6, utilizadas para cabeçalhos de seções no HTML. Sendo <h1> o maior nível e <h6> o menor nível.

As tags H devem ser abertas e fechadas da mesma forma que html e body

Utilizamos estas tags para representar níveis de conteúdo dento do HTML. Para representar títulos costumamos usar h1, h2 para subtítulos, h3 para tópicos, etc.
